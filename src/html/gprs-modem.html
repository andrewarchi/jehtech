U-Blox Commands

UUSIMSTAT=<mode> Configures the +UUSIMSTAT URC presentation. Based on the configuration, the URC is able to report the SIM card initialization status, the phonebook initialization status and the REFRESH active command execution result.
AT&Kx Flow Control Selection: Enable RTS/CTS flow control. Where x is:
  • 0 - Flow control disabled
  • 1 - Normally unused (deals with RTS/CTS)
  • 2 - Normally unused (deals with XON/XOFF)
  • 3 - Enable RTS/CTS flow control
  • 4 - Enable XON/XOFF flow control
  • 5 - Enable transparent XON/XOFF flow control
  • 6 - Enable unidirectional XON/XOFF flow control
AT+UANTR=<antenna-id> Start/set antenna detection. Response is +UANTR <antenna-id> <antenna-load> (in kOhm). A load of -1 represents an open circuit.
AT+UDNSRN Resolve name / IP through DNS
Set +UDNSRN=0,"ip-address" | OK
+UDNSRN=1,"domain" | OK
Get 0,"ip-address or domain" on success,
-1 on error
